Менеджер паролей pass позволит вам сохранить много денег, которые вы могли потратить на RoboForm или LastPass. Видеоинструкция тут.

Вот некоторые команды

Генерируем ключи шифрования

gpg --full-gen-key - Будут сгенерированы публичный и приватный ключи шифрования.
gpg -k - Отобразить публичные ключи.
gpg -K - Отобразить приватные ключи.
pass init uzagal@proton.me - Инициализируем хранилище паролей, указав ключ, которым будут шифроваться наши пароли.
pass insert youtube.com - Добавляем пароль сайта youtube.com в хранилище паролей.
pass - Отобразить записи в хранилище паролей.
pass youtube.com - Расшифровать и показать пароль.
gpg --export -a uzagal@proton.me > pub_uzagal.gpg - Экспортировать публичный ключ и записать его в файл pub_uzagal.gpg.
gpg --export-secret-key -a uzagal@proton.me > sec_uzagal.gpg - Экспортировать приватный ключ и записать его в файл sec_uzagal.gpg.
gpg --delete-secret-keys uzagal@proton.me - Удалить приватный ключ из gpg.
gpg --delete-keys uzagal@proton.me - Удалить публичный ключ из gpg.
gpg --import pub_uzagal.gpg - Импортировать публичный ключ в gpg.
gpg --import sec_uzagal.gpg - Импортировать приватный ключ в gpg.
pass insert facebook -m - Добавить записи в хранилище паролей режиме мультилайн.
pass -c facebook - Копирует пароль для facebook.com в буфер обмена на 45 сек. Через 45 секунд пароль удаляется из буфера обмена.
